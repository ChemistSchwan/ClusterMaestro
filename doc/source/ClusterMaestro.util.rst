rmsd module
-----------

.. attention::
   An about 30\% faster version is available by loading the module *rmsdFAST* instead of *rmsd*.

   *rmsdFAST* is roughly the same code as *rmsd*, but was compiled using Cython, which leads to an performance increase of around 30 %.
